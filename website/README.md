# DJ Sonu — Portfolio Website

A minimal and clean Next.js portfolio website for **DJ Sonu** — a professional DJ and mix artist.

## Features

- 🎵 **Hero section** with DJ Sonu branding and CTA buttons
- 👤 **About section** with bio and stats
- 🎧 **Mixes section** showcasing YouTube mixes
- 📬 **Contact form** for bookings and collaborations
- 🔗 **Social links** — YouTube, Instagram, Facebook
- 📱 Fully **responsive** design (mobile-first)
- ⚡ Built with **Next.js 16 + Tailwind CSS v4 + TypeScript**

## Getting Started

```bash
cd website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deploy

The easiest way to deploy is via [Vercel](https://vercel.com):

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

## Customization

- Update the **mixes** in `app/components/Mixes.tsx` with real YouTube video IDs
- Update the **bio** in `app/components/About.tsx`
- Connect the **contact form** in `app/components/Contact.tsx` to a backend service (e.g. [Formspree](https://formspree.io), [EmailJS](https://www.emailjs.com), or a custom API route)
